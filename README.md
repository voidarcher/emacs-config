# GNU Emacs Configuration Backup 

This is a backup of my Emacs configuration. As of creating this repo, I've been using Emacs
for about an hour and a half, so there's not much here. It does a few basic things:

* Sets the theme to Dracula
* Changes font to Source Code Pro Med, size 13
* Removes the scroll bar
* Makes Emacs open in full-screen
* Adds proper C formatting
* Sets electric pair mode on
